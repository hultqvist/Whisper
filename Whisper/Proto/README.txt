This directory contain the .proto specifications used.

The C# code files are generated by ProtoBuf-C#Gen
https://silentorbit.com/protobuf-csharpgen/
using the following commands:

CodeGenerator.exe Keys.proto Whisper.Proto Keys.cs
CodeGenerator.exe PipeMessages.proto Whisper.Proto PipeMessages.cs

The generated files are then located as follows:

Whisper/ProtocolParser.cs
Whisper/Keys/Keys.*
Whisper/Storing/Pipe/PipeMessages.*
